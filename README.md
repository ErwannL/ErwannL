## ⭐ Orqea — le projet sur lequel je passe le plus de temps

<div align="center">

[![Live](https://img.shields.io/badge/🌐_Live-orqea.dev-7c3aed?style=for-the-badge&labelColor=0d1117)](https://orqea.dev)
&nbsp;
[![Stack](https://img.shields.io/badge/Stack-Node.js_·_React_·_MySQL_·_Docker-blue?style=for-the-badge&labelColor=0d1117)](https://orqea.dev)
&nbsp;
[![Status](https://img.shields.io/badge/Status-Production-22c55e?style=for-the-badge&labelColor=0d1117)](https://orqea.dev)

> **Une alternative à Trello — temps réel, chiffrée, et pensée pour les équipes qui prennent leur data au sérieux.**

</div>

Orqea, c'est un kanban collaboratif qu'on a démarré à zéro avec mon co-fondateur. Aujourd'hui en production sur **[orqea.dev](https://orqea.dev)**, avec :

| | |
|---|---|
| 🔒 | **Chiffrement de bout en bout** des boards (modes `serverOnly` / `twoTierStrict` / `passphraseOnly`) |
| 🌐 | **Temps réel** Socket.IO — drag & drop, comments, notifications instantanées |
| 🔌 | **Intégrations** GitHub · GitLab · Trello · Jira (import de tâches en quelques clics) |
| 🔐 | **OAuth** Google · GitHub · Discord |
| 🤖 | **Agents IA** par carte (prompts custom, automatisations) |
| 🌍 | **i18n** FR / EN sur tout le produit |
| 🐳 | **Déploiement Docker** complet — nginx (front) + Express (back) + MySQL 8, reverse-proxy WSS |

**Stack :** `Node.js (ESM)` · `Express 5` · `React` · `Tailwind CSS` · `MySQL 8` · `Socket.IO` · `JWT` · `Docker Compose` · `nginx`

📬 **Une question, un partenariat, une demande ?** → [contact@orqea.dev](mailto:contact@orqea.dev)

> **Rendre les médicaments accessibles 24h/24, sans file d'attente.**

Mon **projet de fin d'études Epitech (EIP)** — deux briques :

- 🏧 **Un distributeur automatique** en vitrine de pharmacie (comme un ATM) — délivre des médicaments avec/sans ordonnance via scan carte vitale + carte d'identité

**Stack :** `TypeScript` · `Python` · `Java` · `JavaScript`
